# Heart-Key Resonance Protocol (HKRP) - دليل الاستخدام المُحدّث

## 📋 نظرة عامة

نظام محاكاة لبروتوكول المصادقة الحيوي-الترددي الذي يعامل القلب كمُولد إشارة متماسكة للتحقق من شروط التوافق قبل السماح بأي تفاعل مع الطبقة الجينية.

**✨ الإصدار الجديد يتضمن:**
- 🔬 **نموذج استجابة جينية متكامل** يحاكي تأثير التماسك القلبي-الدماغي على التعبير الجيني
- 🧬 **7 جينات مُحاكاة** مرتبطة بالاسترخاء والمناعة
- 📊 **تقارير جينية مفصلة** تظهر تغيرات التعبير الجيني في الوقت الفعلي

## 🧬 المكونات الجينية الجديدة

### 1. `GeneProfile` - ملف التعريف الجيني
يمثل خصائص كل جين:
- `gene_id`: المعرف الفريد (مثل NR3C1, BDNF)
- `gene_name`: الاسم العلمي
- `function`: الوظيفة البيولوجية
- `baseline_expression`: مستوى التعبير الأساسي [0, 1]
- `max_expression`: الحد الأقصى للتعبير [0, 1]
- `sensitivity`: حساسية الجين لإشارة التماسك [0, 1]
- `response_time_ms`: زمن الاستجابة بالميلي ثانية
- `coherence_threshold`: عتبة التماسك المطلوبة للتفعيل

### 2. `ExpressionState` - حالة التعبير الجيني
تمثل الحالة الحالية لكل جين:
- `current_expression`: مستوى التعبير الحالي
- `target_expression`: مستوى التعبير المستهدف
- `activation_level`: مستوى التفعيل [0, 1]
- `is_active`: هل الجين مفعّل حاليًا؟

### 3. `GeneticResponseModel` - نموذج الاستجابة الجينية
**الوظائف الرئيسية:**

#### أ. جينات الاسترخاء والتماسك:
| الجين | الوظيفة | التعبير الأساسي | الحد الأقصى | العتبة |
|-------|---------|-----------------|-------------|--------|
| **NR3C1** | مستقبل الجلوكوكورتيكويد - تنظيم التوتر | 0.30 | 0.85 | 0.70 |
| **BDNF** | عامل التغذية العصبية - النمو العصبي | 0.40 | 0.90 | 0.75 |
| **OXTR** | مستقبل الأوكسيتوسين - الترابط | 0.35 | 0.80 | 0.72 |
| **SIRT1** | سيرتوين 1 - إصلاح الخلايا | 0.25 | 0.75 | 0.80 |
| **FOXO3** | عامل النسخ - الحماية التأكسدية | 0.30 | 0.70 | 0.78 |

#### ب. جينات المناعة:
| الجين | الوظيفة | التعبير الأساسي | التأثير | العتبة |
|-------|---------|-----------------|---------|--------|
| **IL10** | إنترلوكين-10 - مضاد التهابات | 0.20 | ↑ يزيد | 0.75 |
| **TNF** | عامل نخر الورم - التهابي | 0.50 | ↓ ينخفض | 0.70 |

## 📐 المعادلات الرياضية للاستجابة الجينية

### معادلة التعبير الجيني:
```
E(t) = E_baseline + (E_max - E_baseline) × S × G_open × C_normalized
```

حيث:
- `E_baseline`: التعبير الأساسي
- `E_max`: التعبير الأقصى
- `S`: حساسية الجين
- `G_open`: درجة فتح البوابة
- `C_normalized`: التماسك الطبيعي فوق العتبة

### حساب التماسك الطبيعي:
```
C_normalized = (C_actual - C_threshold) / (1.0 - C_threshold)
```

## 🎯 أمثلة متقدمة

### مثال 1: معالجة إشارة مع تقرير جيني

```python
from hkrp_engine import BioSignal, HKRPEngine, SafetyConfig

# إنشاء المحرك
engine = HKRPEngine()

# إنشاء إشارة حيوية مخصصة
signal = BioSignal(
    hrv_sync=0.88,        # تماسك قلبي عالي
    eeg_phase_lock=0.82,  # قفل طور دماغي جيد
    breath_rhythm=0.79    # تنفس منتظم
)

# معالجة الإشارة
result = engine.process_signal(signal)

# عرض النتائج الأساسية
print(f"التماسك: {result['coherence']:.3f}")
print(f"التشابه: {result['similarity']:.3f}")
print(f"حالة البوابة: {result['gate_state']}")

# عرض النتائج الجينية
genomic = result['genomic_summary']
print(f"\n🧬 متوسط تعبير جينات الاسترخاء: {genomic['avg_relaxation_expression']:.3f}")
print(f"🛡️ تعديل المناعة: {genomic['avg_immunity_modulation']:.3f}")
print(f"📊 الجينات النشطة: {genomic['active_genes_count']}/{genomic['total_genes']}")

# عرض تفاصيل كل جين
for gene_id, state in result['gene_states'].items():
    print(f"\n{gene_id}:")
    print(f"  التعبير: {state['expression']:.3f}")
    print(f"  التفعيل: {state['activation']:.1%}")
    print(f"  نشط: {state['is_active']}")
```

### مثال 3: تكوين مخصص للتجارب

    num_iterations=30,

    noise_levels=[0.01, 0.03, 0.05, 0.08]

print(f"\nمتوسط تعبير الاسترخاء: {report['avg_relaxation_expression']:.3f}")
print(f"متوسط تعديل المناعة: {report['avg_immunity_modulation']:.3f}")

## 📊 تفسير النتائج الجينية

### مستويات التعبير:
- **0.0 - 0.3**: تعبير منخفض (حالة أساسية)
- **0.3 - 0.6**: تعبير متوسط (تفعيل جزئي)
- **0.6 - 1.0**: تعبير عالي (تفعيل كامل)

### مؤشرات الأداء:

#### جينات الاسترخاء:
- **↑ زيادة التعبير**: مؤشر إيجابي على استجابة الاسترخاء
- **→ ثبات التعبير**: تماسك غير كافٍ للتفعيل
- **مثال**: BDNF عند 0.75 يعني دعم قوي للنمو العصبي

#### جينات المناعة:
- **IL10 ↑**: زيادة المضادات الالتهابية (إيجابي)
- **TNF ↓**: انخفاض المؤشرات الالتهابية (إيجابي)
- **التوازن المثالي**: IL10 عالي + TNF منخفض

## 🔍 حالات الاستخدام

### 1. البحث العلمي النظري
```python
# دراسة العلاقة بين التماسك والتعبير الجيني
engine = HKRPEngine()
coherence_levels = [0.70, 0.75, 0.80, 0.85, 0.90, 0.95]

for target in coherence_levels:
    results = engine.run_simulation(num_iterations=10, coherence_target=target)
    report = engine.get_session_report()
    print(f"التماسك {target}: تعبير استرخاء = {report['avg_relaxation_expression']:.3f}")
```

### 2. تحسين المعايير
```python
# إيجاد العتبة المثلى للتفعيل الجيني
best_threshold = None
best_activation = 0

for threshold in [0.70, 0.75, 0.80, 0.85]:
    config = SafetyConfig(tau_safe=threshold)
    engine = HKRPEngine(config=config)
    results = engine.run_simulation(num_iterations=20, coherence_target=0.85)
    report = engine.get_session_report()

    activation = report['avg_active_genes']
    if activation > best_activation:
        best_activation = activation
        best_threshold = threshold

print(f"العتبة المثلى: {best_threshold} (تفعيل: {best_activation:.1f} جين)")
```

## ⚠️ ملاحظات أخلاقية وتنفيذية هامة

1. **المحاكاة فقط**: هذا النظام هو نموذج نظري للمحاكاة ولا يتصل بأي أنظمة بيولوجية حقيقية.

2. **البيانات الوهمية**: جميع الإشارات والجينات المُستخدمة هي بيانات مُولَّدة إحصائياً لأغراض البحث النظري.

3. **العزل الأخلاقي**: لا يوجد اتصال حقيقي بأي نظام جيني أو عصبي بيولوجي.

4. **الأبحاث المستقبلية**: أي تطبيق حقيقي يتطلب:
   - موافقة مستنيرة من المشاركين
   - مراجعة لجان الأخلاقيات الطبية
   - امتثال لمعايير السلامة الحيوية
   - إشراف فرق طبية متخصصة

5. **النموذج الجيني المبسّط**: النموذج الحالي يستخدم معادلات مبسطة ولا يعكس التعقيد الكامل للتنظيم الجيني الحقيقي.

## 📚 الأسس النظرية

### المعادلات الرياضية الأساسية

1. **دالة التماسك**:
   ```
   C(t) = α·HRV_sync(t) + β·EEG_phase_lock(t) + γ·breath_rhythm(t)
   حيث α + β + γ = 1
   ```

2. **الاستجابة الجينية**:
   ```
   E_gene(t) = E_base + (E_max - E_base) × sensitivity × gate_open × coherence_excess
   ```

3. **التنشيط الجيني**:
   ```
   activation = |E_current - E_baseline| / |E_max - E_baseline|
   gene_is_active if activation > 0.1
   ```

4. **بوابة التعبير المشروط**:
   ```
   G_open(t) = σ( V(t) · C(t) - τ_safe )
   σ(x) = 1 / (1 + e^(-8x))
   ```

## 🔧 استكشاف الأخطاء

### المشكلة: جميع الجينات غير نشطة
**الحل**:
- زيادة التماسك المستهدف (`coherence_target`)
- خفض عتبة الأمان (`tau_safe`) في `SafetyConfig`
- تقليل مستوى الضوضاء في المحاكاة

### المشكلة: تفعيل التراجع الآمن بشكل متكرر
**الحل**:
- تقليل `delta_coherence` لجعل النظام أقل حساسية
- زيادة استقرار مصدر الإشارة

### المشكلة: تعبير جيني منخفض رغم التماسك العالي
**الحل**:
- التحقق من حساسية الجينات (`sensitivity`)
- زيادة درجة فتح البوابة عبر تحسين التشابه الطيفي
