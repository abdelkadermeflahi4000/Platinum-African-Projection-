# Heart-Key Resonance Protocol (HKRP) - دليل الاستخدام

## 📋 نظرة عامة

نظام محاكاة لبروتوكول المصادقة الحيوي-الترددي الذي يعامل القلب كمُولد إشارة متماسكة للتحقق من شروط التوافق قبل السماح بأي تفاعل مع الطبقة الجينية.

---

## 🚀 التشغيل السريع

```bash
python hkrp_engine.py
```

سيقوم هذا بتشغيل محاكاة كاملة بـ 15 محاولة افتراضية.

---

## 🧩 مكونات النظام

### 1. `BioSignal` - الإشارة الحيوية المركبة
تمثل البيانات الخام المُلتقطة من:
- `hrv_sync`: تزامن معدل ضربات القلب (0-1)
- `eeg_phase_lock`: قفل الطور الكهربائي الدماغي (0-1)
- `breath_rhythm`: انتظام التنفس (0-1)

### 2. `CoherenceSampler` - عينة التماسك
**الوظيفة**: حساب درجة التماسك اللحظية
```python
C(t) = α·HRV_sync(t) + β·EEG_phase_lock(t) + γ·breath_rhythm(t)
```

### 3. `ResonanceMatcher` - مطابق الرنين
**الوظائف**:
- تحويل التماسك إلى طيف فوتوني افتراضي
- حساب التشابه مع القالب الآمن
- حساب إنتروبيا شانون للطيف

### 4. `ZKBioValidator` - مُصادق المعرفة الصفرية الحيوية
**الوظيفة**: التحقق من شرطين:
- التشابه ≥ τ_safe (عتبة الأمان)
- الإنتروبيا ≤ H_max (الإنتروبيا القصوى)

### 5. `ConditionalGate` - البوابة المشروطة
**الوظيفة**: حساب درجة فتح البوابة الجينية
```python
G_open(t) = σ(V(t) · C(t) - τ_safe)
```

### 6. `SafetyRollback` - التراجع الآمن
**الوظيفة**: مراقبة الشروط الحرجة وتفعيل التراجع الفوري عند:
- معدل تغير التماسك سالب بشدة (dC/dt < -δ)
- الإنتروبيا تتجاوز الحد الحرج

## ⚙️ تكوين معايير الأمان

```python
from hkrp_engine import SafetyConfig, HKRPEngine

config = SafetyConfig(
    tau_safe=0.80,         # عتبة الأمان الدنيا [0.75-0.85]
    h_max=2.5,             # الإنتروبيا القصوى (bits)
    delta_coherence=0.15,  # معدل التغير الحرج للتماسك
    response_timeout_ms=250 # مهلة الاستجابة القصوى
)

engine = HKRPEngine(config=config)
```

## 📊 حالات البوابة

| الحالة | الوصف | درجة الفتح |
|--------|-------|-----------|
| `CLOSED` | مغلقة تماماً | < 0.1 |
| `OPENING` | في حالة فتح | 0.1 - 0.5 |
| `OPEN` | مفتوحة | ≥ 0.5 |
| `ROLLBACK` | تراجع آمن | 0.0 |
| `LOCKED` | مقفلة (حالة طارئة) | 0.0 |

## 🔍 مثال متقدم: معالجة إشارة مخصصة

```python
from hkrp_engine import BioSignal, HKRPEngine, SafetyConfig

# إنشاء المحرك
engine = HKRPEngine()

# إنشاء إشارة حيوية مخصصة
signal = BioSignal(
    hrv_sync=0.88,        # تماسك قلبي عالي
    eeg_phase_lock=0.82,  # قفل طور دماغي جيد
    breath_rhythm=0.79    # تنفس منتظم
)

# معالجة الإشارة
result = engine.process_signal(signal)

# عرض النتائج
print(f"التماسك: {result['coherence']:.3f}")
print(f"التشابه: {result['similarity']:.3f}")
print(f"الإنتروبيا: {result['entropy']:.3f} bits")
print(f"حالة البوابة: {result['gate_state']}")
print(f"درجة الفتح: {result['open_degree']:.3f}")
print(f"المصادقة: {'✅ ناجحة' if result['validated'] else '❌ فشلت'}")
```

## ⚠️ ملاحظات أخلاقية وتنفيذية هامة

1. **المحاكاة فقط**: هذا النظام هو نموذج نظري للمحاكاة ولا يتصل بأي أنظمة بيولوجية حقيقية.

2. **البيانات الوهمية**: جميع الإشارات المُستخدمة هي بيانات مُولَّدة إحصائياً لأغراض البحث النظري.

3. **العزل الأخلاقي**: لا يوجد اتصال حقيقي بأي نظام جيني أو عصبي بيولوجي.

4. **الأبحاث المستقبلية**: أي تطبيق حقيقي يتطلب:
   - موافقة مستنيرة من المشاركين
   - مراجعة لجان الأخلاقيات الطبية
   - امتثال لمعايير السلامة الحيوية
   - إشراف فرق طبية متخصصة

## 📚 الأسس النظرية

### المعادلات الرياضية الأساسية

1. **دالة التماسك**:
   ```
   C(t) = α·HRV_sync(t) + β·EEG_phase_lock(t) + γ·breath_rhythm(t)
   حيث α + β + γ = 1
   ```

2. **التحويل الطيفي**:
   ```
   Φ(λ, t) = ∫ C(τ) · K(λ, τ) dτ
   K(λ, τ) = نواة غاوسية معتمدة على التماسك
   ```

3. **التحقق من المصادقة**:
   ```
   V(proof, template) = 1 إذا كان:
     - sim(proof, template) ≥ τ_safe
     - entropy(proof) ≤ H_max
   ```

4. **بوابة التعبير المشروط**:
   ```
   G_open(t) = σ( V(t) · C(t) - τ_safe )
   σ(x) = 1 / (1 + e^(-8x))  # سيجمويد سلس
   ```

5. **دالة التراجع**:
   ```
   R(t) = 1 إذا كان:
     - dC/dt < -δ  (تدهور سريع في التماسك)
     - H(t) > H_critical  (إنتروبيا حرجة)
   ```
