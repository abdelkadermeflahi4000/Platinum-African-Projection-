# 🗺️ خارطة طريق المشروع | Project Roadmap

## المرحلة 1: التأسيس (Foundation) - 🟡 قيد التطوير

### الأهداف:
- [x] بناء النواة الأساسية (Reality Engine)
- [x] تعريف حدود "الإغلاق" الأولي للواقع
- [ ] كتابة اختبارات شاملة للنواة
- [ ] توثيق API الأساسي

### المكونات التقنية:
```
src/
├── reality_engine.py      ✅ مكتمل
├── perspective_engine/    🟡 قيد التطوير
│   ├── __init__.py
│   ├── camera_system.py
│   └── viewport_manager.py
├── temporal_closure/      ⚪ قادم
│   ├── __init__.py
│   ├── loop_simulator.py
│   └── break_conditions.py
└── core/                  🟡 جزئي
    ├── __init__.py
    ├── bounding_box.py
    └── metrics.py
```

### المعايير الفلسفية:
- تعريف "الحدود الإدراكية" كنظام إحداثيات رياضي
- نمذجة "الإغلاق الفكري" كدالة عتبة (Threshold Function)
- تمثيل "كسر الحاجز" كانتقال طوري (Phase Transition)

---

## المرحلة 2: التوسيع (Expansion) - ⚪ قادم

### الأهداف:
- [ ] إضافة طبقات (Layers) مخفية لا تظهر إلا بتغيير الزاوية
- [ ] بناء نظام منظور ديناميكي يستجيب لحركة المستخدم
- [ ] تطوير خوارزميات الكشف عن "الأراضي الأخرى"

### المكونات الجديدة:
```
src/
├── hidden_layers/
│   ├── etheric_layer.py      # الطبقة الأثيرية
│   ├── temporal_layer.py     # الطبقة الزمنية
│   └── parallel_earth.py     # الأرض الموازية
├── perspective_shift/
│   ├── angle_detector.py
│   ├── awareness_mapper.py
│   └── revelation_trigger.py
└── webxr_integration/
    ├── xr_session.py
    └── 360_renderer.py
```

### التقنيات المضافة:
- **Three.js**: للعرض ثلاثي الأبعاد في المتصفح
- **WebXR API**: لدعم نظارات VR/AR
- **WebSocket**: للاتصال اللحظي بين الخادم والواجهة

### المعايير الفلسفية:
- إثبات وجود "أراضٍ أخرى" خلف المنظور الحالي
- تحويل "كسر الإغلاق" من مفهوم مجرد إلى تجربة حسية
- ربط التغير الإدراكي بالتغير الهندسي في الفضاء الافتراضي

---

## المرحلة 3: التحرر (Liberation) - ⚪ مستقبلي

### الأهداف:
- [ ] ربط المشروع بنظارات الواقع الافتراضي (VR)
- [ ] الانتقال من الإدراك المحدود إلى منظور الـ 360 درجة الكامل
- [ ] بناء تجربة غامرة متعددة المستخدمين

### المكونات المتقدمة:
```
src/
├── vr_experience/
│   ├── headset_integration.py
│   ├── motion_tracking.py
│   └── immersive_renderer.py
├── multiuser/
│   ├── shared_reality.py
│   ├── sync_protocol.py
│   └── collective_awareness.py
└── ai_enhancement/
    ├── awareness_ml.py
    └── adaptive_difficulty.py
```

### التقنيات المطلوبة:
- **OpenXR**: للتوافق مع جميع أجهزة VR
- **WebGPU**: لعرض عالي الأداء
- **Machine Learning**: لتكييف التجربة مع مستوى وعي المستخدم

### المعايير الفلسفية:
- الانتقال من الإدراك الفردي إلى الجماعي
- استكشاف فكرة "الوعي المشترك" في الأكوان المتوازية
- تحويل التجربة من محاكاة إلى أداة تأمل فلسفي عملي
